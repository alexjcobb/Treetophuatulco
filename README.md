# TreeTop Villa GitHub Pages Site

## Files included
- `index.html` – homepage
- `stay.html` – property details and amenities
- `gallery.html` – photo gallery
- `area.html` – location and nearby highlights
- `reviews.html` – guest review page
- `faq.html` – common questions
- `contact.html` – contact details and booking handoff
- `book.html` – booking summary page
- `styles.css` – site styling
- `script.js` – small enhancement script
- `assets/` – logo and image files
- `.nojekyll` – ensures GitHub Pages serves the site as plain static files

## Publish to GitHub Pages
1. Create a new **public** repository in GitHub.
2. Upload all files from this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and the **/(root)** folder.
6. Save.
7. Wait for GitHub Pages to publish the site.

## Optional custom domain
In **Settings → Pages**, add your custom domain after the site is live.
Then update DNS with your registrar.
